Como configurar a cor do GlyshTerm

A configuração do emulador disponibiliza a modificação de cores, que não é complicado 

na aba de cores use o sistema de cores RGB(R: Red G: Green B: Blues) para modificar a cor do GlyshTerm

Exemplo:
        
  
        border_color = 160, 160, 160
  

Neste exemplo ele vai definir a cor da borda que é o padrão do GlyshTerm(o mesmo que está no site)

Caso queira que eu explique de uma forma mais direta:
    
        border_color = <R>, <G>, <B>
    

R = Red(vermelho)
G = Green(verde)
B = Blue(azul)

também se aplica ao background_color e foreground_color
