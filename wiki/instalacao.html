Como instalar o GlyshTerm

O GlyshTerm possuí um script automatizado de instalação chamado de glyshterminstall 

Para instalar no Windows, é só executar no cmd(por precaução):
            
            curl -O https://glysh-term.vercel.app/downloads/glyshterminstall.bat && ./glyshterminstall.bat
            
(tenha o curl instalado e funcionando)

Já no Linux é só executar:
            
                curl -O https://glysh-term.vercel.app/downloads/glyshterminstall.sh && ./glyshterminstall.sh
            
(tenha o curl instalado e funcionando)
